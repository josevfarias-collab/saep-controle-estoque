# Sistema de Controle de Estoque para Equipamentos Eletrônicos (SAEP)

Este repositório contém a entrega oficial da solução prática desenvolvida para a avaliação do SAEP 2026. O projeto consiste em um sistema completo de gestão e monitoramento de insumos e componentes eletrônicos.

## 👤 Desenvolvedor
* **Analista:** José Victor Feitoza Farias


---

## 🛠️ Tecnologias e Infraestrutura Utilizadas
* **Ambiente de Execução:** Node.js v24.15.0 (com JavaScript ES6+)
* **Framework Web:** Express.js
* **Gerenciamento de Sessão:** `express-session` (Garantindo rotas internas protegidas)
* **Algoritmo de Ordenação:** Bubble Sort (Implementação manual no servidor para listagem A-Z)
* **Banco de Dados:** MySQL Server 8.0 / Community Edition
* **Sistema Operacional de Desenvolvimento:** Microsoft Windows 10 / 11 Professional (64-bits)

---

## 📋 Pré-requisitos
Antes de iniciar o projeto, certifica-te de que tens instalado na tua máquina:
1. [Node.js](https://nodejs.org/) (Versão estável recomendada)
2. [MySQL Server](https://dev.mysql.com/downloads/mysql/) (Versão 8.0 ou superior)
